**2026.13.2**

**Friends Sidebar**

* Added a new **Same Location** section to the Friends Sidebar.
  This shows all friends who are in the same instance, making it easier to find your friend group and see where they are.

**Changes**

* Updated how **Current Avatar** appears in user profiles.
  Previously, the Avatar ID was shown in the info section below the biography. It is now displayed above **Badges** using the new **vrcn-info** style.
  You will now see an image and the avatar name in that info card. Clicking it will open the Avatar Modal, where you can clone the avatar.
* VRCNext now keeps only the latest 10 log files in `VRCNext/Logs` instead of saving an unlimited number of logs.

**Fixes**

* Fixed an issue that caused the user profile to glitch on first open when someone had no valid avatar.
* Fixed a bug that caused the Friends Sidebar to show **In-World** instead of the actual world name.
* Fixed an issue where your own user profile did not show badge information like other user profiles.
* Fixed an issue introduced after a CSS refactor that caused status colors to appear grey.

**Other Changes**

* General project cleanups
