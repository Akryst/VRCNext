**2026.17.0**

**Kikitan XD**
* Added **Kikitan XD** as a new tool in the sidebar (translator icon).
* Live speech-to-text using the **Groq Whisper API** — no local model required.
* Automatic or manual **source language** detection across 23 supported languages.
* Real-time **translation** to 25 target languages via Groq LLaMA.
* Recognized speech and translation shown side by side in the UI.
* Results are sent directly to the **VRChat OSC Chatbox** (`/chatbox/input`, port 9000).
* Translation and OSC output can each be toggled independently.
* Includes a live **microphone volume bar** identical to Voice Fight.
* All settings (API key, device, languages, toggles) persist across sessions.
* Fully localized in all 6 supported languages (EN, DE, FR, ES, JA, ZH-CN).

**Voice Fight**
* Removed **Mute Talk** toggle — OSC chatbox output is now handled exclusively by Kikitan XD.
